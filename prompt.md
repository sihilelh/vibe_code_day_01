Goal:
Generate a single-file index.html Student Profile Card with a frosted glass (light) glassmorphism theme, including social links, skill tags, and hover animations — written for an intermediate HTML/CSS developer.
Context:
This is a front-end design exercise to build a polished student profile card displaying:

A circular profile image placeholder
Student name (bold, prominent typography)
Stream/course (e.g. Computer Science, Year 2)
Short bio (2–3 sentences)
Skill tags (e.g. Python, UI Design, Data Analysis)
Social link icons (GitHub, LinkedIn, Email)

The card should sit over a soft, light pastel gradient background and use frosted glass styling — a white semi-transparent panel with backdrop-filter: blur(), subtle border, and a light box shadow. Hover effects should animate the card (lift + glow) and the social icons (scale + color shift). Use a modern Google Font (e.g. Poppins or Inter).
Constraints:

Single index.html file — no external CSS or JS files
Must open directly in any browser with no server
Vanilla HTML & CSS only — no frameworks or libraries except Google Fonts CDN
Use CSS custom properties (--variables) for colors and blur values
Add section comments clearly labeling: Layout Structure, Card Styles, Glassmorphism Effect, Typography, Skill Tags, Social Icons, and Hover Animations
Frosted glass must use backdrop-filter: blur(12px–20px) with background: rgba(255,255,255,0.25)
Skill tags must use pill/badge styling with a light tinted background
Social icons can use Unicode symbols or inline SVG — no icon library CDN
